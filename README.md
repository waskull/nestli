## Description

Backend del sistema Helados CALI hecho con [Nest](https://github.com/nestjs/nest) 

## Instalación

```bash
$ npm install
```

## Comandos para Ejecutar la aplicación

```bash
# Desarrollo
$ npm run start

# Desarrollo(modo watch)
$ npm run start:dev

# producción
$ npm run start:prod
```

## Pruebas

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
